### 当前进度（2026.4）

- ✅ iOS MVP v0.1 核心框架已完成
- ✅ 4 个主要 Tab 已实现：
  - 发现（HomeView - 超跑黑橙科技风格首页）
  - 技能市场（MarketView - 分类浏览 + 搜索 + 一键安装）
  - 我的技能（MySkillsView - 执行 + Siri 短语入口）
  - 个人中心（ProfileView - 账号、隐私、开源声明）
- ✅ 统一黑橙超跑科技视觉风格（深黑背景 + 橙红 accents + 玻璃拟态）
- ✅ Shared 组件层完整复用（SkillCardView、GlassPanel、InstallActionButton 等）
- ✅ View + ViewModel 分离，代码结构清晰

### 技术栈

- SwiftUI + App Intents + SiriKit
- Core Data（本地持久化）
- MVVM 架构
- 统一黑橙超跑科技主题

### 如何在本地运行（Mac + Xcode）

1. Clone 项目
2. 用 Xcode 打开 `LamborghiniOS` 文件夹
3. Command + R 运行

（Windows 用户目前可在 Codex 中继续开发代码，等有 Mac 环境后再编译运行）

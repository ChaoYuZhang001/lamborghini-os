### 项目简介
Lamborghini OS 是一个**开源的移动端 AI Agent 操作系统**，运行在 iOS 和 Android 之上。它不是真正的手机 OS，而是一个智能中间层，让普通用户通过**自然语言**就能完成复杂的手机自动化任务。

核心理念：**将手机自动化从“编程”变成“说话”**，并构建全球开发者共同贡献的「技能市场」。

### 核心特性
- 自然语言指令 → 自动执行（Siri / Google Assistant）
- 社区驱动的 Skill Market（一键安装技能）
- 越用越智能（个人 AI 记忆机制）
- 完全开源（MIT）+ Open Core 商业模式

### 快速开始
1. 下载 iOS / Android App（即将上线）
2. 浏览技能市场 → 一键安装
3. 对 Siri 说一句话即可执行

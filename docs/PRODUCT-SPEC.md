## 1. 产品概述

Lamborghini OS 是一个开源的、专注于个人生活场景的 AI Agent 操作系统。**它并非真正的手机操作系统**，而是运行在 iOS 和 Android 系统之上的“智能中间层”。其核心使命是：**将复杂的手机自动化操作，转变为如同说话一样简单的自然语言指令**。

通过连接社区智慧与 AI 大模型，Lamborghini OS 让每一位用户都能拥有一个越用越懂自己、能够跨应用执行任务的专属数字管家。项目致力于降低手机自动化的使用门槛，并构建一个由全球开发者共同贡献的“技能（Skill）”生态。

## 2. 市场背景与竞品分析

### 2.1 行业趋势
2026 年，科技行业正经历从“AI 对话”到“AI 执行”的关键转折。全球消费者电子领域的 AI Agent 市场规模已达数亿美元，中国市场年增长率预计超过 70%。高通 CEO 将 2026 年定义为“AI 代理之年”，预示着一个巨大的市场机遇。

### 2.2 主要竞品与参照系

| 竞品/参照系 | 类型 | 核心特点 | 我们的差异化机会 |
| :--- | :--- | :--- | :--- |
| **OpenClaw** | 开源 Agent 框架 | 强大的多 Agent 协作，丰富的渠道集成 | 更专注于**个人生活场景**，而非服务器/工作流管理 |
| **Hermes Agent** | 开源 Agent 框架 | 自进化能力，长期记忆 | 结合其**学习能力**，并落地到**移动端** |
| **Shortcuty / RoutineHub** | 快捷指令社区 | 海量的用户分享内容 | 引入 **AI 自然语言理解**，实现“技能找人” |
| **iScreen / Widgetsmith** | 桌面美化/小组件 | 巨大的用户基础，成熟的商业模式 | 我们的自动化能力提供的是**更深层的效率价值** |
| **AutoGLM / 豆包手机助手** | 手机 AI 助手 | 跨应用操作（如点外卖、订票） | 强调**开源与社区生态**，让用户掌握自主权 |

### 2.3 市场痛点
- **自动化门槛高**：系统自带的快捷指令等工具学习成本高，普通用户难以掌握。
- **信息孤岛**：用户的自动化创意分散在各处，缺乏一个统一、智能的聚合平台。
- **AI 能力浅层化**：现有手机 AI 助手多停留在问答，缺乏跨应用执行复杂任务的能力。

## 3. 产品定位与差异化价值

### 3.1 核心定位
**Lamborghini OS = OpenClaw 的社区生态 + Hermes 的自进化理念 + 移动端深度适配**

### 3.2 差异化价值
1.  **开源与社区驱动**：核心框架完全开源，建立透明的“技能市场”，让全球开发者共同贡献。
2.  **自然语言交互**：用户无需学习复杂的编程逻辑，用说话的方式即可创建和执行自动化任务。
3.  **越用越智能**：引入 AI 记忆机制，学习用户习惯，从被动响应升级为主动服务。
4.  **跨平台策略**：优先深耕 iOS 生态（利用快捷指令），中期以 Android 为突破口实现真正的跨应用自动化。

## 6. 用户使用指南

### 6.1 安装与初始化

1.  **下载安装**：从 Apple App Store 或 Google Play 下载 Lamborghini OS 应用。
2.  **权限授权**：首次启动时，应用会请求必要的网络权限。后续根据使用的技能，会按需请求特定权限（如相册访问、日历读写等）。
3.  **同意协议**：阅读并同意《用户协议》和《隐私政策》。

### 6.2 核心操作流程

#### 6.2.1 浏览与获取技能

1.  打开应用，进入底部 **「市场」** 标签。
2.  通过顶部搜索栏输入自然语言描述，例如：“我想把截图快速拼成长图”。
3.  浏览搜索结果，点击感兴趣的技能卡片，查看详情。
4.  确认技能描述和所需权限后，点击 **「获取此技能」**。
5.  该技能会自动添加到 **「我的技能」** 列表中。

#### 6.2.2 执行技能

- **方式一 (应用内执行)**：进入「我的技能」页面，找到对应技能，点击右侧的 **「执行」** 按钮。
- **方式二 (Siri 语音执行)**：
    1. 在「我的技能」页面，点击技能卡片进入详情，选择 **「添加到 Siri」**。
    2. 录制一句个性化语音短语，例如：“嘿 Siri，拼接截图”。
    3. 此后，只需对 iPhone 说出该短语，即可触发技能执行。

#### 6.2.3 创建并分享技能 (开发者/高级用户)

1.  使用系统自带的「快捷指令」App 创建一个新的自动化流程。
2.  确保该快捷指令功能完善、测试无误。
3.  在 Lamborghini OS 的「个人中心」选择 **「分享我的技能」**，填写描述、分类并上传该快捷指令文件。
4.  提交后，技能将进入审核队列，通过后即可在市场展示。

### 7.3 iOS 平台特殊适配说明

由于 iOS 系统的“沙盒”安全限制，Lamborghini OS 在 iOS 平台上的跨应用自动化能力主要通过 **Siri 快捷指令** 实现。任何技能的执行，本质上都是触发一个或多个快捷指令的运行。App 本身不进行任何模拟点击或越狱操作，完全符合 App Store 审核指南。

## 8. 隐私与安全

### 8.1 数据收集原则
- **最小必要**：仅收集为实现功能所必需的数据。
- **本地优先**：用户的个人习惯、使用记录等敏感数据默认存储在设备本地。
- **透明可控**：所有数据收集行为均在隐私政策中明确告知，并取得用户授权。

### 8.2 权限使用说明
- **网络权限**：用于访问技能市场、同步数据。
- **Siri 与搜索**：用于支持语音唤醒和执行技能。
- **相册/日历/通讯录等**：仅在执行特定技能时，经用户明确授权后临时调用。

### 8.3 开源安全承诺
- 核心代码在 GitHub 完全开源，接受社区监督。
- 定期进行安全审计，并及时修复漏洞。
- 鼓励安全研究人员通过官方渠道负责任地披露漏洞。

## 12. 常见问题（FAQ）

**Q1：Lamborghini OS 是手机操作系统吗？需要刷机或 Root 吗？**
A：不是。它是一个普通的手机 App，你从官方应用商店下载安装即可，无需任何特殊操作。

**Q2：为什么 iOS 版本的功能感觉比 Android 少？**
A：由于 iOS 系统的“沙盒”安全机制，App 无法像在 Android 上那样通过无障碍服务直接模拟点击操作。iOS 版本的自动化执行能力主要依赖系统官方的「快捷指令」框架，因此部分需要跨应用复杂操作的功能受限。我们会持续探索合规的替代方案。

**Q3：我可以自己开发技能并上传到市场吗？**
A：当然可以！我们欢迎并鼓励所有开发者贡献自己的创意。请查阅我们的《技能开发指南》，按照规范制作并提交你的技能。

**Q4：使用这个 App 会泄露我的隐私吗？**
A：我们高度重视用户隐私。你的个人习惯数据默认保存在本地，仅在执行必要的云端功能（如同步）时，经你明确授权后上传。我们的核心代码开源，接受所有人的审查。

**Q5：如果兰博基尼公司投诉，项目会下架吗？**
A：我们有完善的应对预案。一旦收到正式的法律通知，团队将立即启动品牌更名流程，更换项目代号和视觉标识，以确保项目本身能够长期存续和发展。技术的灵魂不会因名字改变而消失。
